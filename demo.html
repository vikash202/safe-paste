üìñ How It Works

  1. Intercepts Paste: Content script captures paste events across all websites
  2. Detects Format: Automatically identifies JSON, YAML, or plain text
  3. Pattern Matching: Uses regex to find sensitive data patterns
  4. Smart Replacement: Replaces with realistic dummy values
  5. Seamless Insert: Inserts sanitized content naturally
